# rpop

Pop value from the right of a list.

## Syntax

See the [List Operations documentation](/api/list/lpush) for detailed syntax and examples.

## Related Functions
- `ramdb_db_fnc_lpush`: Push element to the left of the list
- `ramdb_db_fnc_rpush`: Push element to the right of the list
- `ramdb_db_fnc_lpop`: Removes and returns the first element(s) from the beginning of the list
- `ramdb_db_fnc_rpop`: Removes and returns the last element(s) from the end of the list
- `ramdb_db_fnc_lindex`: Returns a specific element from the list
- `ramdb_db_fnc_lrange`: Returns a range of elements from the list
- `ramdb_db_fnc_lset`: Replaces a specific element in the list
- `ramdb_db_fnc_lrem`: Removes a specific element from the list
- `ramdb_db_fnc_llen`: Returns the length of the list
- `ramdb_db_fnc_linsert`: Inserts a new element before or after a specific element from the list
- `ramdb_db_fnc_ltrim`: Trims the list to contain only a specific range of elements
