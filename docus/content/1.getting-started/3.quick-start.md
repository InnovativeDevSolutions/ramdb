# Quick Start

This guide will get you using ArmaRAMDb in your Arma 3 missions in just a few minutes.

## Basic Usage

### Storing Data

Store a simple value:

```sqf
["playerName", "John Doe"] call ramdb_db_fnc_set;
```

Store player data using their Steam ID:

```sqf
[getPlayerUID player, [name player, getPos player, getAllGear player]] call ramdb_db_fnc_set;
```

### Retrieving Data

Get a value:

```sqf
private _data = ["playerName"] call ramdb_db_fnc_get;
// Returns: "John Doe"
```

Get player data:

```sqf
private _playerData = [getPlayerUID player] call ramdb_db_fnc_get;
// Returns: ["John Doe", [x, y, z], [...gear...]]
```

### Context Mode

ArmaRAMDb supports "context mode" where an empty key `""` automatically uses the caller's Steam ID:

```sqf
// These are equivalent:
["", "myData"] call ramdb_db_fnc_set;
[getPlayerUID player, "myData"] call ramdb_db_fnc_set;
```

## Hash Tables (Player Data)

Hash tables are perfect for storing structured player data.

### Store Player Info

```sqf
// Store multiple fields for a player
["", [
    "name", name player,
    "money", 5000,
    "level", 10,
    "position", getPosASL player
]] call ramdb_db_fnc_hmset;
```

### Get Specific Field

```sqf
private _money = ["", "money"] call ramdb_db_fnc_hget;
// Returns: 5000
```

### Get All Fields

```sqf
private _allData = [""] call ramdb_db_fnc_hgetall;
// Returns: ["name", "John Doe", "money", 5000, "level", 10, "position", [x,y,z]]
```

### Increment Values

```sqf
// Add money
["", "money", 500] call ramdb_db_fnc_hincrby;

// Add experience (float)
["", "experience", 15.5] call ramdb_db_fnc_hincrbyfloat;
```

## Lists (Event Logs)

Lists are great for logs, queues, and ordered data.

### Add Items

```sqf
// Add to start of list
["serverEvents", "Player joined: " + name player] call ramdb_db_fnc_lpush;

// Add to end of list
["serverEvents", "Server saved at " + str time] call ramdb_db_fnc_rpush;
```

### Get Items

```sqf
// Get specific index
private _event = ["serverEvents", 0] call ramdb_db_fnc_lindex;

// Get range (0 to 9)
private _recentEvents = ["serverEvents", 0, 9] call ramdb_db_fnc_lrange;

// Get all items (0 to -1)
private _allEvents = ["serverEvents", 0, -1] call ramdb_db_fnc_lrange;
```

## Multiplayer Usage

In multiplayer, database operations should run on the server:

### From Server

```sqf
// Direct call (already on server)
["playerStats", someData] call ramdb_db_fnc_set;
```

### From Client

```sqf
// Send to server
["playerStats", someData] remoteExec ["ramdb_db_fnc_set", 2, false];
```

### Return Data to Client

```sqf
// Client requests data with callback
[
    getPlayerUID player,  // Key
    player,               // Target (return to this player)
    "myFunction",         // Function to call with data
    true                  // Use call (not spawn)
] remoteExec ["ramdb_db_fnc_get", 2, false];

// Define the callback function
myFunction = {
    params ["_data"];
    hint format ["Received: %1", _data];
};
```

## Persistence

### Save to Disk

```sqf
// Manual save
[false] call ramdb_db_fnc_save;  // No backup

// Save with backup
[true] call ramdb_db_fnc_save;   // Creates backup
```

::callout{icon="i-lucide-info" color="secondary"}
**Auto-save**: If enabled in config.xml, the database automatically saves at configured intervals.
::

### Load from Disk

```sqf
// Load the main database file
["@ramdb/data.rdb.gz"] call ramdb_db_fnc_load;

// Load a specific backup
["@ramdb/backups/data_20260116_150000.rdb.gz"] call ramdb_db_fnc_load;
```

## Complete Example: Player Persistence

Here's a complete example of saving and loading player data:

```sqf
// ===== Save player on disconnect =====
addMissionEventHandler ["PlayerDisconnected", {
    params ["_id", "_uid", "_name", "_jip", "_owner", "_idstr"];
    
    // Store player data using Steam ID as key
    [_uid, [
        "name", _name,
        "position", getPosASL (objectFromNetId _id),
        "loadout", getUnitLoadout (objectFromNetId _id),
        "money", player getVariable ["money", 0],
        "lastSeen", systemTimeUTC
    ]] call ramdb_db_fnc_hmset;
    
    // Save to disk
    [true] call ramdb_db_fnc_save;
}];

// ===== Load player on connect =====
addMissionEventHandler ["PlayerConnected", {
    params ["_id", "_uid", "_name", "_jip", "_owner", "_idstr"];
    
    // Check if player data exists
    private _exists = [_uid] call ramdb_db_fnc_exists;
    
    if (_exists > 0) then {
        // Load player data
        private _data = [_uid] call ramdb_db_fnc_hgetall;
        
        // Parse data
        private _playerName = _data select (_data find "name" + 1);
        private _position = _data select (_data find "position" + 1);
        private _loadout = _data select (_data find "loadout" + 1);
        private _money = _data select (_data find "money" + 1);
        
        // Apply to player
        private _player = objectFromNetId _id;
        _player setPos _position;
        _player setUnitLoadout _loadout;
        _player setVariable ["money", _money, true];
        
        systemChat format ["Welcome back, %1! Balance: $%2", _playerName, _money];
    };
}];
```
